#### Стек
* React/Redux
* SCSS
* Webpack
* PropTypes
* Jest/Enzyme
* Travis
* ES6+, async/await
* SemiStandard, ESLint, Stylelint

#### npm scripts
* ```npm run watch``` — запускаем webpack-dev-server
* ```npm run test``` — тестируем
* ```npm run build``` — билдим

* ES6+, Jest/Enzyme
* Pixel Perfect
* Отзывчивая вёрстка
* Компонентный подход — ключевые компоненты независимы друг от друга
* Линтёры для JS, SCSS
